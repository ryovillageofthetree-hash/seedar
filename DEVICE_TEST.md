# Chromebook 実機テストチェックリスト

種まき支援AR を Chromebook で動作確認するためのチェックリストです。
授業実践の前に、すべての項目をクリアしているか確認してください。

## 0. 準備物

- [ ] Chromebook（Chrome 最新安定版）
- [ ] 印刷した Hiro マーカー（50mm × 50mm）— `assets/markers/print.html` から印刷
- [ ] テスト用プランター（size_S または size_L）
- [ ] 黒土または鉢底石（マーカーが見やすい色のもの推奨）

## 6. 既知の制約

仕様書 §8.3 にある通り：

- 暗所での動作は保証外
- マーカーが汚れたり折れたりすると認識精度が下がる
- 複数のマーカーを同時に画面内に映した場合の挙動は未検証

## トラブルシューティング

### マーカーが認識されない
- マーカーが 50mm × 50mm で印刷されているか定規で測定
- マーカーが折れたり汚れたりしていないか
- 教室の照明を明るくする
- マーカーをカメラから 30〜60cm 程度離す（近すぎる／遠すぎるの両方で認識しにくくなる）

### 円柱の位置がずれる
- マーカーの上辺がプランター長辺と平行に貼られているか確認
- マーカーが浮いたりシワになったりしていないか
- プランターの内寸が仕様書の値（38×14 または 60×20cm）と一致しているか確認

### Chrome がカメラを起動できない
- URL が `https://` で始まっているか
- 他のアプリ（ビデオ会議など）がカメラを占有していないか
- Chromebook の設定 → 詳細 → プライバシーとセキュリティ → サイトの設定 → カメラ
- 学校管理の Chromebook の場合、組織ポリシーでカメラが制限されていないか管理者に確認

### 動作が重い／カクつく
- 他のタブを閉じる
- 別の Chromebook で試して端末固有の問題か切り分け
- size_L 点まき（108 個の円柱）が最も重いケース。これがカクつくなら他の組み合わせも試す
